# hello.js.org
### HelloJS community page


####Development
----------

**Make sure you have ```hellojs.dev``` added into your ```hosts``` file**

    127.0.0.1   hellojs.dev


**Clone repository**

    git clone git@github.com:hello-js-org/hello-js-org.github.io.git hellojs/
    cd hellojs/


**Install dependencies**

    npm install


**Start development**

    gulp development -p


####Production
----------

**Clone repository**

    git clone git@github.com:hello-js-org/hello-js-org.github.io.git hellojs/
    cd hellojs/
    
**Install dependencies**

    npm install
    
**Run production task**

    gulp production -p
